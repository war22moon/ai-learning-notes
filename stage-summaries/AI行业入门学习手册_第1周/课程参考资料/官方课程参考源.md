# 官方课程参考源

## Anthropic Claude / Claude Code

用于学习 Prompt Engineering、Agent、MCP、上下文工程、工具调用、评估、Claude Code 工程协作、CLAUDE.md、Skills、Hooks、Subagents 等方法。

## OpenAI Codex

用于学习 AI 编程协作、多 Agent 软件交付、终端/IDE/云环境工作流、代码评审、CI/CD 和团队研发效率。

## 智谱 GLM

用于学习国产大模型平台、长上下文、Function Call、结构化输出、MCP、国产 MaaS 与工程场景交付。

## 使用原则

官方课程作为一线厂商教学样板，用于帮助 100 天课程保持产业前沿和真实产品语境，但输出时必须转换成用户熟悉的售前、产品和交付语言。
